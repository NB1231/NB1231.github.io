练好联系群众基本功
发布于：2019-01-28 12:46:11 标签：/ xylt / 访问：

　　密切联系群众是党的优良传统，脱离群众是党面临的最大危险。学习贯彻习近平新时代中国特色社会主义思想和党的十九大精神，必须持之以恒练好密切联系群众的基本功。
　　站稳政治立场，赢得群众拥护。人民立场是中国共产党的根本政治立场，是马克思主义政党区别于其他政党的显著标志。习近平总书记在看望江西贫困群众时满怀深情地说：“我们是人民的勤务员，帮你们跑事的。”在实际工作中，必须始终把群众放在心中最高位置，坚持“手机公开、大门打开、脚步迈开”，贴近群众，宣传党的方针政策，解决实际困难，化解矛盾纠纷。把群众满意不满意、答应不答应、高兴不高兴作为衡量工作得失的标准，同群众“一块苦、一块干、一块过”，厚植干群鱼水情深，确保群众始终拥护党、跟随党，听党话、跟党走。
　　畅通联系渠道，引导群众参与。新时代新形势，迫切需要创新互动形式，畅通互动渠道。应当建立健全完善党员干部联系群众常态化、领导干部下访机制化、政务工作公开化等公众参与的制度机制。用好12345呼叫中心、融媒体中心、政务平台、手机报、微平台等新媒体技术，特别是通过群众参与度高、认可性强、覆盖面广的新媒体平台，主动公开、积极宣传，与群众互动，让群众知晓，引导群众主动参与。例如，通过群众喜闻乐见的形式，公开政务时事，传播正能量，既可以有效避免谣言产生，也可以更广泛地让群众主动参与进来。
　　坚持干在实处，确保群众信任。“空头支票”损害党的形象，伤害群众感情。向群众承诺的事情，不论困难多大，必须一抓到底。必须按照习近平总书记的要求做到件件有落实、事事有回音，让群众看到变化、得到实惠。坚持把财政支出的大部分用于改善民生，不论是加强乡村路桥等基础设施建设，把农村建成秀美乡村，还是生态、就业、扶贫、医疗、社保、棚改等百姓急切盼望的项目，都要落到实处。聚焦建档立卡贫困户就业创业，发挥“第一书记”、驻村工作队作用，打好脱贫攻坚战。
　　持续恒久努力，凝聚群众力量。伟大的事业需要伟大的精神，伟大的精神托举伟大的梦想。如果群众工作要得100分，那么群众认不认可，愿不愿追随，就是“1”，而如果这个“1”没有了，所有的结果就是“0”。所以必须自始至终坚持走群众路线，不断加强精神文明建设，大力弘扬社会主义核心价值观，广泛开展党风、家风、作风建设，使社会主义核心价值观与传统优秀文化结合，使向善向上思想潜移默化深入人心，坚守党领导下的群众精神文明阵地，提升满足群众需求的实效性和精准度，增强广大群众跟党走的向心力，把群众牢牢地团聚在党的周围。
