新视域下如何坚定文化自信
发布于：2019-01-28 05:26:57 标签：/ xylt / 访问：

　　坚定文化自信是习近平新时代中国特色社会主义思想的鲜明符号。党的十九大报告指出，我国社会主要矛盾已经转化为人民日益增长的美好生活需要和不平衡不充分的发展之间的矛盾。进入新时代，我国社会主要矛盾的变化是关系全局的历史性变化，坚持以习近平文化思想为指导，从新矛盾视域重新审视坚定文化自信的研究，具有重要的学理意义和实践价值。
　　文化供给侧结构关系
　　需求端。当今中国，从需求端看，“人民群众期盼更丰富的精神文化生活”，集中体现为既昂扬向上，又多姿多彩的主旋律、正能量文化，突出表象在三方面。一是期盼习近平新时代中国特色社会主义思想武装创新。人民群众期盼着习近平新时代中国特色社会主义思想以更加大众化生活化的方式武装头脑，期盼着习近平文化思想以更具穿透力凝聚力的内核引领时代，指引人民圆满实现民族复兴的伟大中国梦。二是期盼中华优秀传统文化繁荣辉煌。广大群众期盼中华传统文化能够适应时代变化，科学地进行创造性转化、创新性发展，更加深入地走进生活，更好地育人化人，不断铸就中华文化新辉煌。三是期盼社会文化环境健康向上。广大群众期盼着更加清朗的网络空间、更加优秀的文化精品、更加丰富的文化服务、更加高尚的道德氛围、更加诚信的社会风气、更加彻底地消除负面文化现象，打造富足和谐的“文化小康”。
　　供给端。从供给端看，我国文化还存在三个发展不平衡不充分的突出问题。一是文化样态需要从“高原”向“高峰”转变。新矛盾已经指明我们必须转变“高原过剩”“高峰缺失”的态势，不断推进文化样态精品化高端化个性化发展，充分发掘借鉴创新文化中的时代因素，积极打造全面满足个人需求的多层级文化样态。二是文化功能需要从硬性向韧性转变。当下我国对文化功能的运用经常采用硬性的灌输和运用，强自取折，很多领域还要根据文化发展规律，充分释放文化“润物无声”的使用价值，做到摧刚为柔。三是文化发展需要从融合向创新转变。当前我们的文化融合发展迅速，展现了中华文化海纳百川的大气魄，但是融合只是文化发展的一个环节，最难点还是要结合实践需要，满足政治经济需求、符合人民群众需用，强化文化创新。
　　坚定文化自信的优势条件
　　习近平新时代中国特色社会主义思想，为坚定文化自信提供了强大思想武器。文化自信是习近平新时代中国特色社会主义思想的重要组成部分，是习近平文化思想的基本内核。习近平新时代中国特色社会主义思想既在理论层面为今后如何坚定文化自信提出根本思想遵循，也在实践层面为今后如何坚定文化自信提出正确路径指引，是新时代我们坚定文化自信最为强大的思想武器。
　　以习近平同志为核心的党中央对文化工作高度重视，为坚定文化自信提供了强劲支持力量。党的十八大以来，以习近平同志为核心的党中央面对世界范围内各种思想文化日趋激烈的交流交融交锋，从坚持和发展中国特色社会主义、实现中华民族伟大复兴中国梦的政治高度，以高度的文化自信、文化自觉、文化担当，科学规划和全面铺开了新时代大背景下中国特色社会主义文化建设。
　　社会稳步健康发展，为坚定文化自信提供了强壮基础支撑。社会稳步健康发展，为坚定文化自信提供重要物质基础和思想先决条件。我国经济快速发展、社会的持续稳定必将直接促进文化领域的繁荣发展，正面影响弘扬和传播当代中国文化创新成果的文化自觉与自信，使国人的文化自信获得日益丰富的内涵、愈加扎实的根基、更为厚实的底气。
　　一系列方针政策出台落地，为坚定文化自信提供了强力制度指引。党的十八大以来，党中央印发了一系列重要文件，直指新时代社会主义文化强国建设中一系列方向性、全局性、战略性的重大问题，对文化建设作出全面安排、提出明确要求，形成了全面系统、科学完整的工作体系和工作框架，为推进文化建设指明了前进方向、提供了根本遵循，形成了坚定文化自信的重要制度保障。
　　全党全国学习宣传贯彻习近平新时代中国特色社会主义思想和党的十九大精神，为坚定文化自信提供了强盛战略契机。学习宣传贯彻习近平新时代中国特色社会主义思想和党的十九大精神，为坚定文化自信提供了强盛的战略契机，学习宣传贯彻党的十九大精神的过程，就是以习近平文化思想这一当代中国文化创新成果引领大众文化和社会思潮的过程，就是我们坚定文化自信、实现中华文化繁荣发展的过程。
　　坚定文化自信的路径选择
　　从传统文化滋养中提升文化认同。中华传统文化是我们文化发展的母体，是中华民族最独特的性格气节和最根本的文化基因。时代革新、矛盾转化，需要把弘扬优秀传统文化和发展现实文化有机统一起来、紧密结合起来，在继承中发展，在发展中继承，进一步增强中国共产党文化自觉和文化自信，进一步提升马克思主义理论传统厚度和历史深度。
　　从执政规律把握中增进文化自省。立足当代中国现实，结合当今时代条件，发展面向现代化、面向世界、面向未来的，民族的科学的大众的社会主义先进文化，更好地展现中华文化独特魅力，形成与我国综合实力相适应的国家文化软实力，推动社会主义精神文明和物质文明协调发展。
　　从社会发展实践中释放文化伟力。牢牢掌握意识形态工作领导权，推进马克思主义中国化时代化大众化，把习近平新时代中国特色社会主义思想贯彻到事业发展全过程、体现到改革发展各方面。大力培育践行社会主义核心价值观，切实加强思想道德建设，提高全社会文明程度。推动文化事业文化产业发展，为满足群众美好生活新期待提供更丰富、更广泛的精神文化食粮。
　　从世界发展格局中彰显文化担当。当前，我国正日益走近世界舞台的中央。当代中国，日益彰显着大国文化自信、文化自觉和文化担当。通过中国文化的传播、中国声音的传递、中国精神的传承、中国力量的传扬、中国形象的传颂，切实增强中国文化号召力和凝聚力，提升中国话语的能见度和影响力，把中国故事讲精彩、传得远。
