对无职党员实施“一编三定”
发布于：2019-01-28 12:37:35 标签：/ xylt / 访问：

　　对农村无职党员实施“一编三定”，是基层党组织发挥首创精神，结合形势需要，引导农村无职党员发挥作用进行的有益探索，对于进一步增强基层党组织的政治引领和服务功能，更好地密切党与群众的血肉联系起到了很好的促进作用。
　　“一编”即编员进组，就是把认领相同岗位的党员编成一个组，选出一名组长，由一名“两委”干部进行分包，指导和带领该组党员开展工作、发挥作用。“三定”即定岗位、定责任、定奖惩。定岗位，就是根据全村发展实际、群众所思所盼和党员自身能力素质，合理设置工作岗位，由无职党员进行认领，党支部研究确定；定责任，就是按照岗位特点确定工作职责，明确目标任务，界定履岗责任；定奖惩，就是定期对无职党员履岗情况进行考评，并结合考评情况进行奖惩，增强无职党员立足岗位创先争优、发挥作用的主动性和积极性。
　　具体实践中，主要从四个方面进行了有益探索：一是科学设岗，以岗定责，强化党员主体意识，创造履职尽责平台。针对一些农村基层党员，有名无岗、想干事无平台的现象，积极实行设岗定责、编员进组，让无职党员想干事有平台、有机会。二是加强教育，严格管理，提升党员能力素质，夯实履职尽责基础。坚持把加强党员教育管理、提升党员素质能力作为一项基础工作来抓，着力解决无职党员“不愿、不能、不会”发挥作用的问题。三是加强教育，严格管理，提升党员能力素质，夯实履职尽责基础。在“一编三定”工作实践中，为保证无职党员履好职、尽好责，发挥作用，探索实施组长负责、两委分包制。四是定期评岗，严格奖惩，倒逼党员责任压力，激发履职尽责干劲。探索建立“定期评岗、动态管理”工作机制，通过定期汇报、定期评议、定期奖惩、定期调整，充分调动党员履职尽责的积极性，保持无职党员队伍的生机和活力。
　　（作者系中央党校县委书记研修班学员、河南省许昌市建安区委书记）
