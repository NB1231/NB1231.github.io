以扩大内需促进质量变革
发布于：2019-01-28 12:36:03 标签：/ xylt / 访问：

　　改革开放40年，我国经济取得了举世瞩目的成绩，人民群众的生活水平由温饱实现了小康，居民消费水平迅速提升，当前正处于扩大内需和消费升级的历史机遇期。
　　内需已成为经济稳定运行的“压舱石”。据国家统计局核算，2008—2017年内需对经济增长的年均贡献率达到105.7%、超过100%，2018年一季度对经济增长贡献率达到109.1%。得益于居民收入保持较快增长、消费升级势能持续增强，中国近14亿人口造就了巨大的国内市场，内需成为拉动我国经济发展的决定性力量。
　　消费升级已成为经济转型发展的“助推器”。随着经济持续发展和居民收入持续改善，我国已造就了全球最大规模的中产阶级人口，涌现了“首选品质而非价格”的买单者，追求品质更好、体验更佳、服务更到位，正由大众消费向个性化、多元化、差别化、品牌化消费转变，消费升级“井喷式”爆发，已成为高质量发展的重要推动力量。
　　必须用好扩大内需和消费升级的叠加机遇。面对“逆全球化”和全球贸易保护主义，中国作为一个大国，国内市场广阔、经济梯度大，具有很强的经济韧劲和回旋余地。聚焦扩大内需和消费升级所创造的“世界级”市场，加快供给侧结构性改革，不仅能够有效应对国际贸易的不确定性，也能够满足人民日益增长的美好生活需要，推动我国经济的可持续发展。
　　实施“三品”专项行动。聚焦先进制造和中高端消费类产品。一是增品种。支持企业围绕加快产品升级换代、发展中高端产品开展技术革新和改造，培育开发一批制造精品。二是提品质。实施“标准化+先进制造”，围绕主导产业、重点产品组织质量比对和会商会诊，打造一批国家级、省级质量标杆企业。三是创品牌。组织开展区域品牌培育试点，打造制造业单项冠军企业，提升品牌的知名度和美誉度。
　　创新全过程质量管控。一是加强全员、全方位、全过程质量管理，引导企业加快质量设计理念、质量控制技术、质量管理模式变革，提高质量在线监测、在线控制和产品全生命周期质量追溯能力。二是推行个性化定制、柔性化生产，鼓励企业优化功能设计、提高产品扩展性、耐久性、舒适性等质量特性，满足绿色环保、可持续发展、消费友好等需求。三是促进协同制造和协同创新，鼓励各类市场主体和高水平检验检测服务平台，整合生产组织全过程要素资源，纳入共同的质量管理、标准管理、合作研发管理等，实现质量水平整体提升。
　　大力弘扬工匠精神。充分发挥质量品牌建设中“人”的作用。一方面，围绕主导产业和特色集群发展需求，引导建设由政校企参加的职教联盟，推进产教深度融合，逐步建立政府、企业、劳动者三方共同投入的高技能人才培养机制，加大急需、紧缺高技能人才的培养力度；另一方面，大力弘扬专业专注、精益求精的工匠精神，鼓励骨干龙头企业广泛建设“技能大师工作室”，组织职工参加各类竞赛比武活动，塑造名匠品牌，营造尊重劳模、崇尚工匠的社会氛围。
　　（作者系中央党校中青二班学员、河南省工业和信息化委员会副主任）
