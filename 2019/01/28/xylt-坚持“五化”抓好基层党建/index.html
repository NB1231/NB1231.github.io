坚持“五化”抓好基层党建
发布于：2019-01-28 05:31:52 标签：/ xylt / 访问：

　　必须坚持常态化。抓基层党建并不是一朝一夕的事情，要把抓基层打基础作为长远之计和固本之策，一任接着一任干，一张蓝图绘到底，要常磨常新、坚持不懈、久久为功，决不能打“突击战”、搞“一阵风”。
　　必须坚持制度化。通过建立健全党委书记抓党建责任清单、问题清单、任务清单、领办项目和公开承诺等工作制度，层层压实支部抓党建的主体责任。要建立完善考核体制机制，量化工作绩效，把作风扎实、苦干实干、干出成绩的干部选出来用起来，切实将党建成果转化为发展成果。
　　必须坚持信息化。善于积极运用“互联网+党员教育”“智慧党建”等信息化手段，对“主题党日”“村务管理”等实行全流程记录、大数据分析，提升党员教育管理服务信息化、精准化水平，解决组织生活和监督管理“限时、限地、限人”的成本高、效率低的难题，进一步加强基层组织的凝聚力和执行力。
　　必须坚持实效化。要把党建工作与经济社会发展紧密结合起来，坚持工作推进项目化，建立红色细胞责任网格，善于借助党的建设推动工作落实，做到党的建设与改革发展稳定同向发力、相互促进。推动基层党建全面过硬，更好实现经济社会健康持续发展。
　　必须坚持严肃化。要进一步健全村级党组织工作规则，理顺“两委”关系，明确树立村党支部的领导核心地位。要完善基层党建巡察、第三方调查等监督制度，综合运用“反馈、通报、约谈、问责、函询、入档”等手段，切实加大基层党建责任落实不到位的问责力度。
