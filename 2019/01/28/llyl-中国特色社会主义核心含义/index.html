中国特色社会主义核心含义
发布于：2019-01-28 12:11:16 标签：/ llyl / 访问：

　　
　　改革开放以来党的理论创造和理论建树，集中表现为以邓小平理论为开端且在实践中不断发展的中国特色社会主义。中国特色社会主义的核心含义至少包括四方面内容。
　　第一，全面地突破苏联模式。从邓小平在党的十二大开幕词中提出走自己的路，建设有中国特色的社会主义开始，它的最直接、最现实、最深刻的含义，就是一定要摆脱苏联模式，解放思想，发展生产力，走向改革开放。
　　第二，中国现代化目标的重新定位、重新调整。从邓小平理论开始，以富民为先，以民生为重，以不断提高人们的生活水平为最高目的，对中国现代化的目标作了重新的说明和定位。一个完全着眼于人民生活的概念，其中包含着对现代化目标新的深刻的理解，那就是我们所要争取的现代化，是完全为了不断提高人民生活，不断使人民的日子好起来的现代化。
　　第三，主动地进入世界现代文明的主流，充分地吸收人类的文明成果。用邓小平的话来说，就是三个面向：面向现代化，面向世界，面向未来。中国特色社会主义越来越鲜明地表现出它是主动参与世界文明进程的社会主义。中华文化历来具有对外来文化足够的包容、吸纳和融合能力。这种开放精神极其可贵。
　　第四，破除长期存在的对马克思主义的教条式理解和附加在马克思主义名下的错误观点。中国特色社会主义突破了僵化的苏联模式，同时意味着它回到了马克思的理论逻辑，贯彻这个理论逻辑而得出了一些全新的结论。不要简单地把它看作是对马克思主义的突破和否定，例如从计划经济转向市场经济，很多人以为好像这是突破了马克思的理论，这样认识问题是不正确的。
