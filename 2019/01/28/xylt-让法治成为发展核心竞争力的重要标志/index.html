让法治成为发展核心竞争力的重要标志
发布于：2019-01-28 05:27:27 标签：/ xylt / 访问：

　　全面建成小康社会目标的实现，不仅需要法治的保障，更需要法治的引领。面对新形势新任务，只有进一步重视、加强法治的作用，才能在未来发展中保持竞争优势。
　　在推进经济更高质量发展上发挥法治核心竞争力。法治对经济具有引领、规范和保障作用，法治是现代市场经济的重要特征，成熟的市场经济需要健全的法治与之相呼应。从我国这些年经济体制改革发展历程上看，法治建设的一个突出表现体现在促进、保障经济发展上。今后经济要持续深入健康发展，必须发挥法治的核心竞争力，发挥法治在保护市场竞争、维护市场秩序、激发市场活力中的功能作用，通过法治更加公平公正地配置市场资源。针对经济结构调整、企业转型升级、科技创新发展中的新情况新问题，需要及时跟进法律服务手段，加大对创业投资、公平竞争、技术创新的依法保护力度，积极助力大众创业、万众创新。充分依靠法治手段调整经济秩序，保护市场主体创新活力，维护公平竞争的市场环境。当前经济社会迅速发展，新产业、新业态以及新商业模式、投资模式层出不穷，法治保障、法治供给应当及时跟上。
　　在推进人民生活更加幸福上发挥法治核心竞争力。在当前语境下，我们所追求的“生活更加幸福”的内涵是丰富的、外延是广泛的，不仅要让老百姓口袋鼓起来，还要为老百姓提供安居乐业的条件、平安和谐的环境、合法权益的保障等。从根本上解决民生问题和发展民生事业，法治必须先行，必须发扬尚法的民生理念，把民生纳入法治化的轨道，强化民生领域的立法、执法、司法等法治环节，把法治的价值、理念、思维、方法等贯穿到民生建设的全过程。更加注重依法保障权利，加快完善体现权利公平、机会公平、规则公平的地方法规体系，保障公民人身权、财产权、基本政治权利等各项权利不受侵犯，保障公民经济、文化、社会等各方面权利得到落实，不断加大民生法治保障力度。更加注重增强人民群众安全感，在更高水平上推进平安建设，依法从重惩处各类严重刑事犯罪，严厉惩治发生在群众身边、严重损害群众权益的电信诈骗、非法集资以及盗抢骗、黄赌毒、危害食品药品安全等案件，积极促进社会安定有序。要更加注重对权力的依法规制，及时回应民众对政府行为的关注和期盼，借助现代网络媒体技术推进政务公开，通过法律手段真正将权力关进制度的笼子，实现让权力在法治的框架内规范运行。
　　在推进生态环境更加优美上发挥法治核心竞争力。从本质上讲，要实现绿色发展，切实贯彻落实绿色发展的战略部署，使生态环境治理真正取得实效，必须运用法治手段促进和保障绿色发展健康运行，建立健全环境治理法治体系，实现绿色发展法治化。当前，生态环境是全面建成小康社会的突出短板，整治当前环境治理中的突出难题，必须综合施策推动生态环境质量实现根本性好转，尤其需要最大限度地发挥法治在推进生态文明建设中的功能作用。进一步完善资源环境保护法律法规，为推进生态文明建设提供立法支持。加大环境执法力度，依法查处环境违法行为，有效落实网格化环境监管体系，坚决淘汰落后产能，以最有力的环境执法解决突出环境问题。贯彻恢复性司法和预防性司法理念，充分发挥环境公益诉讼职能作用，依法维护社会公众环境权益，为实现天更蓝、地更绿、水更清提供有力司法保障。
　　在推进文化发展更加繁荣上发挥法治核心竞争力。当前，我国文化领域正在发生广泛而深刻的变革，推动文化大发展大繁荣既具备许多有利条件，也面临一系列新情况新问题，如一些领域道德失范、诚信缺失，一些社会成员人生观、价值观扭曲等等。这就迫切需要充分发挥法治在解决道德领域突出问题中的独特功能，引导人们自觉履行法定义务、社会责任、家庭责任。一方面，在文化建设过程中，需要积极培育社会主义法治文化，弘扬社会主义法治精神，增强全社会学法尊法守法用法意识，把社会主义核心价值观贯彻到法治建设的具体实践中，落实到立法、执法、司法、普法和依法治理的各个方面。依靠依法行政、司法裁判、法治宣传等手段大力弘扬仁爱、明礼、诚信、孝悌、和谐的传统文化，促进社会公德、职业道德、家庭美德和个人品德建设，努力培育良好的社会风尚。针对当前诚信缺失问题，应当充分发挥法治在诚信建设中的重要作用，抓紧建立健全征信系统，加大对失信行为惩戒力度，在全社会广泛形成守信光荣、失信可耻的氛围。另一方面，则需要发挥法治对文化发展的促进和保障作用，加大对拥有自主知识产权、弘扬民族优秀文化的产业支持力度，依法惩处侵权行为，保护知识产权人合法权益。同时，注重网络文化监督和管理，加强网上舆论引导，依法惩处传播有害信息行为，严厉打击网络违法犯罪，有效净化文化环境。
　　在推进社会治理更加完善上发挥法治核心竞争力。在全面建成小康社会进程中，面对社会矛盾纠纷复杂多变的趋势，迫切要求按照法治观念和法治逻辑去观察、分析和解决社会治理中遇到的问题，提升社会治理体系和治理能力现代化水平。善于运用健全的法治管住任性的权力，防止因不依法办事、侵犯群众合法权益引发社会矛盾。养成在法治轨道上主张权利、解决纷争的习惯，推动把利益诉求纳入法治化轨道。针对群众反映强烈的突出问题，善于以典型个案依法处理为突破口，给有关行业、群体确立规则，发挥好执法司法在规范社会行为、引领社会风尚中的重要作用。健全完善矛盾纠纷化解的体制机制，依法构建多元化纠纷解决机制，引导群众理性表达诉求，推动建立从源头上有效减少和化解矛盾纠纷的工作机制，形成工作合力。
　　（作者系中央党校省部班学员、江苏省政协副主席）
