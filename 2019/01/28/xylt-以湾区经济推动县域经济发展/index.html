以湾区经济推动县域经济发展
发布于：2019-01-28 12:42:31 标签：/ xylt / 访问：

　　湾区经济是依托港口（群）、背靠广阔腹地发展而成的重要滨海经济形态，是当今国际经济版图的突出亮点，是世界一流滨海城市的显著标志。粤港澳大湾区的提出，为推动湾区经济的发展提供了新的典范。无疑，从县域经济角度看，在我国大力发展湾区经济同样具有十分积极的意义。
　　湾区在地理形态上具有海岸凹入内陆的特征，因此湾区经济不同于一般的沿海经济，具有“拥海抱湾集群”的复合特性，能释放港口、城市、创新、产业等的叠加效应。湾区因其经济结构开放、功能协作紧密、资源配置高效、辐射外溢效应强大，成为主导所在区域经济发展的核心区块。
　　优越的地理环境是湾区经济发展的先决条件。湾区凹入内陆、海岸绵长、腹地广阔，拥有天然良港，能够高效开展海陆交接。国内外各知名湾区，无一不催生于港口，成长于海陆互通。如纽约湾拥有北美洲最繁忙的纽约港和繁华的纽约州、新泽西州等内陆腹地，水运航线、铁路运输线分别多达200多条和14条，为纽约湾集中了美国北大西洋集装箱货运市场运输量的55%。
　　强大的产业集群是湾区经济发展的重要支撑。国内外知名湾区产业形态各异，产业并不一定“姓海”，但必定依托于湾区的地缘优势，顺应时代发展不断衍变升级。如杭州湾紧跟时代步伐调整产业结构，在改造提升传统的轻工、皮革、纺织、服装等产业集群基础上，创新引领、抢抓机遇，电商、软件、信息、文创等新兴产业蓬勃发展，地区生产总值已超越旧金山湾区。
　　集聚的资源要素是湾区经济发展的原生动力。国内外知名湾区大致经历了港口经济、工业经济、服务经济、创新经济等四个阶段，在其经济结构演进过程中，资本、技术、人才等优势资源、创新要素始终保持高度集聚，为湾区经济发展提供了强劲持久的原生动力。如旧金山湾的硅谷拥有斯坦福大学等顶尖学府，集聚了世界各国科技人员100万人以上，其中获诺贝尔奖者就达30人，集聚了全美国1/3以上的风险投资公司和总部风险基金公司，高端要素的集聚使旧金山湾成为引领世界高科技发展的全球创新中心。
　　高效的区域协作是湾区经济发展的关键机制。湾区大多由城市群组成，城市群之间必须协作发展、错位互补才能实现资源的最高效利用。如东京湾区包括东京港、横滨港等在内的7个港口致力破除同质化竞争陷阱，内部运营独立、分工明确，高效利用港口岸线资源，成为世界复合型港口推动湾区经济发展的经典案例。
　　以浙江省乐清市为例，乐清作为全国百强县，历史悠久，经济活跃，具备发展湾区经济的各种有利条件和比较优势。乐清湾位于温州大都市区东北翼、瓯江入海口北侧，是浙江省六大重点湾区之一，环湾地区总户籍人口约187.7万人，陆岸线长约523公里，其中港口岸线长约60.8公里，水深最深达41米，拥有优越的岸线航道条件、充裕的腹地资源、发达的集疏运功能和雄厚的产业支撑。近十几年来，乐清持之以恒推进海洋经济发展，启动实施乐清湾港区大开发，坚持海陆、山海、港城和港产联动，着力构建现代滨海产业体系，为发展湾区经济奠定了坚实基础。
　　结合世界湾区经济发展趋势和我国湾区经济发展现状，从县域层面看，大力发展湾区经济需要采取各项具体措施：
　　打造协同发展、高品质能级的港湾都市区。县域需要主动承接周边大都市区配套功能，推动县域自身转向转型，促进城区融合发展。同时，要围绕提升中心城区首位度，着力增强城市综合品质、功能、形象，特别是强势开展城中村改造，全面消除半城市化面貌，不断优化城市创业居住环境。
　　打造要素集聚、特色彰显的产业转型示范高地。城市中心区加快打造创富港湾小镇，着力建成总部经济、楼宇经济发展示范区和都市核心区。各个县都有自己的经济开发区，这些经济开发区要利用好湾区经济的优势，着力发展外向型、先进型的现代产业体系，使湾区经济区能够成为产业转型的示范高地。
　　打造交通发达、互联互通的开放包容湾区。具备湾区经济发展条件的县域一般都有港口，大力发展县域湾区经济，需要以港口开发开放为引领，以交通路网建设为先导，推进湾区全域、后方腹地和辐射地域的互联互通，提升区域间设施共建共享水平，为县域经济开放发展提供强力支撑。
　　打造绿水青山、宜居宜业的花园式湾区。绿水青山就是金山银山。县域湾区经济开发，要严守生态红线，合理开发低山丘陵等浅山区，筑牢湾区绿色屏障。协同环湾区保护与合理开发利用海洋资源，共同构建沿湾区的生态岸线，用花园式湾区推动县域湾区经济的发展。
　　（作者系中央党校县委书记研修班学员，浙江省乐清市委书记）
