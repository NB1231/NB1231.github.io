防范和化解经济运行风险
发布于：2019-01-28 05:30:28 标签：/ xylt / 访问：

　　今年年初以来，面对异常复杂严峻的国际形势和国内艰巨繁重的改革发展任务，在以习近平同志为核心的党中央坚强领导下，各地区各部门认真贯彻落实党中央的决策部署，坚定践行新发展理念，以深化供给侧结构性改革为主线，主动对标对表高质量发展要求，统筹推进稳增长、促改革、调结构、惠民生、防风险各项工作，国民经济运行总体平稳、稳中有进，三大攻坚战取得明显效果。
　　但在稳步发展的同时，经济和金融市场运行也遇到了一些困难和挑战。一是经济下行压力有所加大。一季度GDP增速保持在6.8%，二季度为6.7%，三季度为6.5%，前三季度增长6.7%，增速总体放缓。二是企业运营遇到一些困难。今年上半年，企业利润仍维持约17%的增长，但上下游之间、不同所有制企业之间和不同规模企业之间存在差异。三是从国际外部环境来看，中美贸易摩擦给我国企业的未来预期带来了一定的负面影响。
　　与此同时，我国金融中一些长期积累的风险逐步显现。具体而言，几个风险点值得我们关注。一是融资结构不尽合理。直接融资比例今年约为5%，处于历史低点，民营和中小企业难以获得资金来源。二是债券市场信用违约风险加剧。截至今年8月，企业债券违约规模较大。三是P2P网络借贷平台问题显现。四是股票质押风险问题显现，少数超高质押比例的股票风险加剧。
　　金融风险的背后，既有其长期的结构性问题，也有短期因素。因此，化解经济和金融风险，应当从短期和中长期分别着手，做到标本兼治。一方面，通过短期政策利多缓解可能的短期风险激化因素；另一方面，通过改革化解长期的金融结构性失衡，坚持在发展中解决问题。第一，加大改革开放力度，稳定社会预期，提振信心，激发作为经济主体的企业的活力。第二，坚持结构性去杠杆的长期方向，把握去杠杆的力度和节奏，把对地方政府、城投平台和国有企业作为去杠杆的重点，加大债转股和不良贷款处置力度。第三，加大股权融资比重，降低杠杆率。加快多层次资本市场体系建设，改善金融结构。第四，加强和改进金融市场监管。既要坚持市场化取向金融改革，为经济社会发展提供动力，又要实现有效和高水平的监管。如何避免“一放就乱、一收就死”的怪圈，是未来我国金融治理最大的挑战和重中之重。近期要采取有效措施，重点关注和防控股票质押风险和P2P网络风险，防范化解局部和系统性风险。
　　（作者：中央党校（国家行政学院）省部级干部进修班防范金融风险课题组）
