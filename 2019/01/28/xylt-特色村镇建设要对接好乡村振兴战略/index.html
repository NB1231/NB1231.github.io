特色村镇建设要对接好乡村振兴战略
发布于：2019-01-28 05:28:34 标签：/ xylt / 访问：

　　在“纳入”政策上下功夫，高位推动特色村镇建设。应当把特色村镇建设作为脱贫攻坚达标验收后最大的帮扶接续政策，作为民族地区实施乡村振兴战略的主要抓手，按照乡村振兴战略实施的时间节点和工作标准，同步编制规划，同步研究部署，同步推进实施，同步检查验收，确保民族地区跟上全国农村的发展步伐，确保如期全面实现乡村振兴的战略目标。
　　在“保护”政策上下功夫，协调推动特色村镇建设。应把“保护第一、抢救第一、特色第一”作为特色村镇建设的基本方针，力避发生“破坏性建设”现象。挖掘和彰显特色村镇的民族文化特色，在“去粗取精、去伪存真”的基础上传承创新。在民族文化保护和利用的具体措施上，一是发挥规划引领的保护作用，二是发挥创意宣传的保护作用，三是发挥活态传承的保护作用。
　　在“发展”政策上下功夫，聚力推动特色村镇建设。实施乡村振兴战略，推进特色村镇建设，必须进一步加大力度制定和落实对民族地区的差别化扶持政策。这就是说，只有按照党的十九大的部署，牢牢把握发展不平衡、不充分这一主要矛盾，不断缩小和消除民族地区的发展差距，民族团结才不至于成为建筑在沙滩上的大厦，才会坚如磐石地屹立在世界的东方。
　　（作者系中央党校厅局班学员、湖南省民委主任）
