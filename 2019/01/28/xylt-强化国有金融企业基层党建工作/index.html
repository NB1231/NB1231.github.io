强化国有金融企业基层党建工作
发布于：2019-01-28 05:28:40 标签：/ xylt / 访问：

　　抓好制度建设，全面落实党建主体责任。从严落实抓党建工作主体责任，明确工作任务，积极构建书记抓党建、班子成员分工负责、职能部门齐抓共管的责任体系。规范党建工作组织保障制度，使执行更有力，管理更有效。强化班子主体责任和领导干部“一岗双责”，明确责任清单。对党建工作各项要求进行量化考核，扎实开展各级班子成员基层党建联系点工作制度。
　　抓好党组织和党员队伍建设，发挥好“两个作用”。坚持基层党组织与经营管理架构同步规划、同步设置、同步管理，以确保党组织作用的发挥。优化领导班子人员结构，将基层党建和人才培养相结合，保持党务工作的先进性和创新性。做好先进典型学习，用行动引导和影响身边的员工，充分调动员工的积极性，使他们努力把工作做好，助推业务发展。
　　抓好工作创新，强化党建工作活力。实施党建共建，实现党建与业务的深度融合。充分利用好党员之家这一有效活动平台，把“党员之家”作为与客户单位共建合作的重要活动场所，与共建单位在“党员之家”中开展共建活动，通过“党建搭台”，推动业务深度合作。构建标准、完善的党建工作信息化平台，从而为基层党建工作的开展创建良好的环境，使党员由被动接受转变为主动参与，为基层党建工作取得突破性的进展提供有效平台。
　　（作者系中央党校中青二班学员、中国建设银行苏州分行行长）
