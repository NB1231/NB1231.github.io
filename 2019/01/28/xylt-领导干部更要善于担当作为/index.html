领导干部更要善于担当作为
发布于：2019-01-28 05:28:13 标签：/ xylt / 访问：

　　新时代呼唤新担当，新担当需要新作为。当前，全党吹响了激励广大干部新时代新担当新作为的冲锋号。对领导干部而言，不仅要敢于担当，更要善于担当、善于作为。
　　切实增强使命感，铸牢担当之“魂”。领导干部的担当作为，首先来自思想上的觉悟，来自理想信念的坚定，来自内在的动力，这是善于担当作为的前提条件。不忘初心、牢记使命，肩负起对党忠诚、为党分忧、为党尽职、为民造福的政治担当，肩负起时不我待、只争朝夕、勇立潮头的历史担当，肩负起守土有责、守土负责、守土尽责的责任担当。坚定理想信念，牢记为人民服务的宗旨，坚守人民情怀，以人民对美好生活的向往作为我们的奋斗目标，真心实意地为老百姓谋幸福。把这份使命转化为做好工作的满腔热情，不管在哪个地方、哪个单位工作，都要热爱那个地方、那个单位的发展和事业，贡献个人的全部力量做好工作，努力改变一地一处的工作面貌。
　　善于把方向，把稳担当之“舵”。领导干部要善于担当作为，就必须提高政治站位，善于观大势、把方向、谋大局。把握政策方向，自觉服从、服务于党和国家事业全局，准确把握党的方针政策，把握新时代提出的新要求，坚决贯彻落实中央各项决策部署。把握工作重点，立足于本地本部门工作实际，对事关全局的重点工作，要有明确的目标定位、工作计划，统筹推进，狠抓落实。突出问题导向，抓住主要矛盾和矛盾的主要方面，扭住改革发展稳定的“牛鼻子”，找准工作的切入点、着力点和突破口，持续用力，久久为功。
　　善于学习，增强担当之“能”。领导干部要做到善于担当作为，就必须严格按照习近平总书记的要求，全面增强学习本领、政治领导本领、改革创新本领、科学发展本领、依法执政本领、群众工作本领、狠抓落实本领、驾驭风险本领，练就善于担当的“真本事”。克服自满和懈怠，自觉加强学习，加强实践，虚心向领导、同事学，向专家、基层和群众学。按照中央关于建设高素质专业化干部队伍要求，加强专业知识、专业能力的学习培训，注重培养专业作风、专业精神，大力提高专业思维和专业素养，做到干一行爱一行、钻一行精一行、管一行像一行。善于总结工作经验，特别是善于总结提升推动工作的好方法和好经验。
　　深入调查研究，夯实担当之“基”。调查研究是谋事之基、成事之道，无论是谋划决策环节，还是执行落实环节，都需要深入地调查研究。调研要勤，要迈开步子，多到一线去调研；对重大问题，要反复多次调研。调研要实，既要到工作开展好的地方去总结经验、提升工作，又要到困难较多、矛盾尖锐、工作做得差的地方去研究问题、推动工作，真正做到听实话、察实情、求实效。调研要深，紧扣工作中需要解决的重点、难点问题，带着问题去、带着思考去，深入开展调查研究，把矛盾和问题搞深摸透，找到解决问题、推动工作的办法。
　　严格依法依规，守住担当之“准绳”。要善于担当作为，必须依法依规，按程序办事。增强法治观念，敬畏法律，带头依法办事，带头遵守法律，坚持在法律授权范围内履行职责，真正做到法无授权不可为，法定职责必须为，坚决杜绝以言压法、以权代法，更加不能徇私枉法。提高法律素养和能力，不断提高运用法治思维和法治方式深化改革、推动发展、化解矛盾、维护稳定的能力，自觉提升依法办事的能力。自觉依法议事决策，依法依规推动工作，做到权限、程序、手段合法，确保各项工作在法治轨道上运行。建立重大决策合法性审查机制，作出决策之前，必须认真听取法制部门的意见，进行严格的法律把关。
　　以上率下，凝聚担当之“合力”。 作为领导干部，善于担当作为还必须切实发挥示范表率作用，以担当带动担当，以作为促进作为。善聚班子之力，认真落实民主集中制，坚持集体领导，发扬民主、善于集中、敢于担责，以开阔的心胸善于听取各方面的意见。强化团队意识，增进班子团结，注重沟通协作，发挥每个成员的工作积极性，依靠团队的力量推动工作。善聚干部队伍之力。坚持严管和厚爱结合、激励和约束并重，鲜明树立重实干重实绩的用人导向。建立健全容错纠错机制，宽容干部在改革创新、推进发展中的失误错误，为担当者担当。善聚群众之力。在思想上尊重群众，在感情上贴近群众，在工作上深入群众，及时倾听群众呼声、回应群众关切、集中群众智慧、调动群众力量。
　　改革创新，增强担当之“效”。领导干部要做到善于担当作为，更要把改革创新放到十分重要而突出的位置。争当改革的支持者。大力支持、鼓励和保护改革实践。在改革创新出现失误时，主动拉一把，鼓励改革，宽容失败，允许试错。争当改革的执行者。按照中央的部署，扎实推进各项改革。吃透改革精神，把握改革方向，正确、准确、有序、协调推进改革。落实改革部署，把中央和上级的要求转化为改革的措施、改革的办法，变为改革的行动。争当改革的探路者。对于方向明确，属于地方和部门可以授权操作的改革，要大胆试、大胆闯。坚持转变转化转型，增创体制机制新优势。不断转变思想观念，转化运用先进经验，推动经济高质量发展，创新社会治理方式，建立充满活力的新体制新机制。
　　（作者系中央党校中青一班学员、广东省茂名市委书记）
