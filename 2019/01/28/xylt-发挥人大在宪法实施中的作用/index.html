发挥人大在宪法实施中的作用
发布于：2019-01-28 05:29:57 标签：/ xylt / 访问：

　　习近平总书记最近在全面依法治国委员会第一次会议上强调：“要全面贯彻实施宪法，在全社会深入开展尊崇宪法、学习宪法、遵守宪法、维护宪法、运用宪法宣传教育活动，弘扬宪法精神，加强宪法实施和监督。”人大作为全面履行宪法和法律赋予职责的国家权力机关，在保证宪法实施、推进宪法宣传教育中担负着十分重要的职责。各级人大要深入学习贯彻习近平总书记的重要讲话精神，重点做好以下几方面工作。
　　加强立法和宪法解释，着力夯实宪法实施制度根基。改革开放特别是党的十八大以来，以宪法为统帅的中国特色社会主义法律体系形成并逐步完善，截至今年8月底，现行有效法律有267件，行政法规有756件，地方性法规12000多件，有效推动了宪法的贯彻实施，促进了依宪治国、依宪执政。全面依法治国，对实现国家和社会生活制度化法制化，提出了新的更高的要求。各级人大应认真学习贯彻新修改的宪法，完善立法规划，提高立法质量，把宪法确定的基本经济制度、基本政治制度和文化建设、社会建设、生态文明建设等方面的规定细化具体化，使宪法精神和法治原则真正落到实处。同时，全国人大常委会要切实履行宪法解释职能，根据实际需求，及时对宪法的相关条款和内容作出解释，确保宪法得到全面准确的实施。
　　加强执法监督工作，着力保障公民基本权利和义务。保证宪法在本行政区域内的遵守和执行，是宪法和地方组织法赋予各级人大及其常委会的重要职责。一是开展法律法规执行情况检查。加大执法监督的力度，每年抓住群众反映强烈的热点问题，开展相关法律法规执法检查，对检查中发现的突出问题要扭住不放，一抓到底，抓出成效。二是加强司法领域的监督。通过听取和审议专项工作报告、督办代表建议、专题询问、“两官”履职评议等形式，加强司法工作监督。对一些地方出现的有案不立、有罪不究，滥用强制措施等问题，要加大跟踪监督力度，增强监督实效，真正让人民群众在每一个司法案件中感受到公平正义。三是切实做好人大信访工作。认真受理人民群众的来信来访，加强信访综合分析，特别是对久拖不决、损害群众切身利益的重要信访件，要及时做好转办督办工作；对涉法涉诉信访案，要加强与司法机关沟通，认真做好移交，发挥好人大信访在公民权利保障中的职能作用。
　　加强备案审查工作，着力维护宪法权威和法制统一。从地方人大来说，要重点强化三方面工作：一是加强主动审查。要从过去被动审查逐步向主动审查为主转变，增强规范性文件备案审查的主导作用，凡同级人民政府作出的决定、命令，下一级人大及其常委会作出的决定决议，都要全部纳入备案审查范围，做到全覆盖。如果存在该报不报的，要及时沟通和督促。二是增强监督刚性。在审查中，一旦发现决定决议或命令存在超越法定权限，限制或者剥夺公民、法人和其他组织合法权利，或者随意增加义务；同法律法规规定相抵触以及其他不适当情形的，要加强沟通协调，及时提出处理整改意见，督促相关部门自行改正。坚决维护宪法的尊严和权威，维护社会主义法制统一。三是充实工作力量。按照建立严密的法治监督体系的要求，加强人大备案审查工作力量配备，把责任心强、法律功底扎实、业务能力过硬的干部，及时选调到备案审查队伍中去，切实提高人大备案审查的质量和水平。
　　加强宪法宣传教育，着力增强宪法意识和法治观念。宪法的根基在于人民发自内心的拥护，宪法的伟力在于人民出自真诚的信仰，只有让宪法家喻户晓、深入人心，宪法实施才能真正成为全体人民的自觉行动。一是全面实施宪法宣誓制度。各级人大应全面落实宪法宣誓制度，凡由人大及其常委会选举、决定任命的国家机关工作人员，就职时都要进行宪法宣誓，不断增强其忠于宪法、维护宪法权威、履行法定职责、接受人民监督的行动自觉。同时，现场宣誓气氛典雅，神圣庄严，具有很强的震撼力和感染力，本身也是宪法教育的鲜活教材。二是组织好国家宪法日相关活动。加强与党委宣传部门和政府司法部门的协同配合，以每年12月4日开展国家宪法日活动为契机，发挥引领主导作用，通过纪念座谈、理论研讨、知识竞赛、法治讲堂、现场咨询等形式，大力宣讲宪法在国家政治生活中的重要地位和作用，讲好宪法实施中的“好故事”。充分依托广播电视、报纸刊物等传统媒体和新闻客户端、微信公众号等新兴媒体，对宪法日活动和重大主题项目及时进行报道，扩大宪法宣传的辐射面和影响力。三是运用宪法教育基地普及宪法知识。有条件的地方要挖掘资源。整合力量，创设宪法和法律教育基地，拓宽普法的渠道和载体。切实担负起宪法宣传教育的职责，充分发挥自身优势，努力营造全社会学习宪法、遵守宪法和维护宪法的浓厚氛围，推动形成办事依法、遇事找法、解决问题用法、化解矛盾靠法的良好法治环境。
