按定律学习理论
发布于：2018-05-28 00:00:00 标签：/ xylt / 访问：

　　短短两个多月的中央党校学习时光，使我深感在马克思主义理论的学习方面收获颇丰。这主要得益于我探索的学习方法——按定律学习理论。应了毛泽东1938年在中央党校的讲话：“学校虽只几个月，但学了一个方法……”
　　“七步”走程序
　　这套程序就是：“啃”原著——习听导读——资料辅读——再读原著——梳理疑问交流——提炼——强记。
　　原著特别是经典一定要下大决心狠“啃”，而且要“细嚼慢咽”。恩格斯曾向别人传授学习经验：“研究原著本身，不会让一些简述读物和别的第二手资料引入迷途。”原著是马克思主义理论本源。读原著是返本归宗，悟原理才能获取真经，学原文才会正本清源，更可以避免碎片化收获。
　　原著特别是经典要精细读、反复读、对照读、讨论读，力求从不同角度、不同层面，更准确全面熟悉透彻地把握其思想内涵和精神实质。毛泽东同志说：“《共产党宣言》我看了不下一百遍。”“每阅读一次，我都有新的启发。”这是常读常新、受益感悟的有力佐证。
　　习听导读，包括课前认真预习教材，自备疑问上课认真听讲，课后下载老师教案。中央党校作为中国共产党的最高学府和红色殿堂，培育集聚了一大批顶尖水平教授，听他们的课常有“听君一席话，胜读十年书”的恍然大悟、一通百通之感。我还充分利用校内几家书店和图书馆资源，通览对马克思主义的解读文章和十八、十九世纪西欧社会史思想史和中华传统哲学类书籍。通过每阶段的小组讨论会和双向交流，进一步促进交流启发、释疑解惑、学学相长、教学相长，并认真梳理问题提交讨论和消化。最后是综合各方所得，对理论进行体系性概括，凝练成精句短文强记，力争今后熟悉运用。
　　“三问”作向导
　　为联结理论与实践，真正撬动思考，要以好问为向导，把理论与实践、历史与现实、工作与个人高度自觉摆进全过程，以实现“学马列要精，要管用”。
　　一是理论本义之问。马克思主义的基本立场、基本观点和基本方法到底是什么？马克思主义的世界观人生观价值观究竟是什么并怎样区分？理论产生的时代背景怎样？有怎样的萌芽、形成、变化过程和体系架构？作者为什么提出这一理论，又是在什么条件下针对什么问题而说？应用了怎样的逻辑分析方法？要力求精准、全面、深刻的发问。
　　二是现实实践之问。理论的发展形态、历史作用和其中的经验教训有哪些？马列理论与马克思主义中国化成果的内在联系是什么？对当今时代的指导意义、价值体现和实践路径在哪里？当前中央制定的方针政策和战略部署的理论源头蕴含着哪个原理？能解决哪些方面问题、对手头工作有何启发？自己该作何改造？要着眼历史、立足当下、具体到人的深问。
　　三是时代考题之问。时代考题集大成，是总纲领、大方向。当前的时代考题是“坚持和发展什么样的中国特色社会主义、怎样坚持和发展中国特色社会主义”有幸身处伟大时代，有幸亲历鲜活丰富中国实践，我们必须自觉担负对习近平新时代中国特色社会主义思想的开拓，自发担当对共产党执政规律、社会主义建设规律、人类社会发展规律的探索。
　　“三法”保成效
　　学理论的根本目的是“涵养正气、淬炼思想、升华境界、指导实践”。在学习阶段要确保有实实在在、可感可观、及时可测的成效。实现路径就是做批注、抄笔记、写体会。马克思说：“养成了对我读过的一切书作摘录的习惯。”他大量的学习笔记和体会后期直接被整理成了经典著作。毛泽东同志的经验也是“不动笔墨不读书”。他读10万字的《辩证法唯物教程》就写下了近1.3万字的批语。所以，只要有一点零星感悟也要做批注，思想的火花弥足珍贵。抄笔记要选取经典精华，并自行高度提炼。写体会则一定要有自己的思考、思想，拿出真知灼见。
　　恩格斯说：“马克思的整个世界观不是教义，而是方法。它提供的不是现成的教条，而是进一步研究的出发点和供这种研究使用的方法。”按定律学习理论，其实就是一种科学有效的学习模式，普适于更宽领域的学习和更多的学人。
　　（作者系中央党校厅局班学员、湖南省委政研室副巡视员）
